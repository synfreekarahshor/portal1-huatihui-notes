# portal1-huatihui-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为页面资料的集中存放与展示用途。  
所有页面均为独立文件，不依赖外部数据库或后端服务。

## 目录说明

```
portal1-huatihui-notes/
├── pages/          # 存放归档的 HTML 页面
├── assets/         # 页面引用的公共资源（如样式、脚本、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或独立 HTML 文件对应一个归档页面。
- `assets/` 目录用于存放多个页面共享的静态资源，避免重复。
- 根目录下的 `index.html` 可作为页面列表或导航入口（如存在）。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，内容可能涉及笔记、工具、演示或其它静态页面。  
每个页面在 `pages/` 下以文件夹形式组织，内部包含该页面所需的全部文件（HTML、CSS、JS 等）。  
页面之间相互独立，无耦合关系。

### 归档规范

- 每个页面应有独立的文件夹，命名简洁明了。
- 页面内所有资源引用建议使用相对路径。
- 如有特殊说明或注意事项，可在对应文件夹内添加 `README.md` 文件。

## 维护说明

本仓库由维护者不定期更新，用于存放或发布新的 HTML 页面。  
- 新增页面时，请遵循上述归档规范。
- 如需修改或删除已有页面，请确保不影响其它页面。
- 欢迎提交 Issue 或 Pull Request 进行讨论与协作。

## 许可

本仓库内容仅供学习与参考，不构成任何形式的承诺或保证。  
具体页面可能附带各自的许可说明，请以页面内标注为准。
